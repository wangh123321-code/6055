### 核心功能交付

| 类别 | 功能 |
|------|------|
| **前端页面** | 社区首页(标签筛选+时间线流+热门排行)、师徒匹配推荐页、个人主页与作品集、收徒帖发布页、求师帖发布页、帖子详情(评论/点赞/收藏)、登录注册(角色选择) |
| **用户系统** | 注册时选角色(手艺人/学习者/爱好者)、JWT认证、资料编辑(标签+地理位置) |
| **帖子系统** | 三种类型(作品/收徒/求师)、标签筛选、分页查询、热门排行 |
| **社交互动** | 点赞(切换)、收藏(切换)、评论(CRUD) |
| **师徒匹配引擎** | 标签向量余弦相似度(权重0.7) + 200km地理位置加权(权重0.3)、每周一凌晨2点定时匹配、推荐结果缓存7天、反馈机制(感兴趣/不合适)优化后续匹配 |
| **部署** | docker-compose(MongoDB + NestJS + Nginx)、Dockerfile多阶段构建、nginx反向代理 |
| **API文档** | Swagger UI，启动后访问 `http://localhost:3001/api` 查看 |

### 设计亮点

- **配色**：中国风红金配色(`#C41E24` + `#D4A84B`)，暖米色背景(`#FFF8F0`)
- **匹配算法**：标签向量化后计算余弦相似度，结合2dsphere地理位置范围查询(200km内加权)，综合得分排序取Top 10
- **标签体系**：蔚县剪纸、陕北剪纸、扬州剪纸、佛山剪纸、满族剪纸等流派标签，贯穿筛选和匹配
